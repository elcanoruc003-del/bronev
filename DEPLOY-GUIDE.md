## 🎯 DEPLOY PLANI

### ADDIM 1: Vercel-ə Deploy (İNDİ)

1. **vercel.com** açın
2. GitHub ilə daxil olun
3. **"Add New Project"** → **bronev** seçin
4. **Environment Variables** əlavə edin:
   ```
   NEXT_PUBLIC_PHONE_NUMBER = 0777670031
   NEXT_PUBLIC_WHATSAPP_NUMBER = 994777670031
   ```
5. **"Deploy"** basın
6. 2-3 dəqiqə gözləyin
7. **Sayt CANLI!** 🎉

### ADDIM 2: Database Əlavə Et (SONRA)

Supabase project tam hazır olduqdan sonra (10-15 dəqiqə):

1. Vercel dashboard → **Settings** → **Environment Variables**
2. Yeni variable əlavə edin:
   ```
   DATABASE_URL = your_supabase_connection_string
   ```
3. **"Redeploy"** basın
4. Database aktiv olacaq!
